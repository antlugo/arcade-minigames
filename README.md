Arcade Minigames

By: Anthony Lugo

A game that faces the player with increasingly harder minigames until they run out of lives.

Made for Arduino UNO R3 (32KB PROGMEM 2KB RAM)

Compile and upload with Arduino IDE [https://www.arduino.cc/en/software/](https://www.arduino.cc/en/software/)

Minimum Memory requirements: 
    32082 bytes PROGMEM, 1538 bytes RAM
    EEPROM unused

Bill of Materials:
    5 buttons
    5 led diodes
    1 128x64 OLED display (u8g2 library compatiable)
    1 joystick
    1 IR reciever with Elegoo IR remote (NEC protocol)
    1 single digit 7-segment display (dot not wired)
    1 SNx4HC595 shift register (exclusively used in output mode)
